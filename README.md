# gachonShop
Node.js 사용해 쇼핑몰 홈페이지 만들기

# 테이블 정보

### person 테이블
 사용자 정보 저장(PRIMARY KEY: loginid)

### merchandise 테이블
 상품 정보 저장(PRIMARY KEY: mer_id)

### code_tbl 테이블
 상품 분류 정보 저장(PRIMARY KEY: main_id, sub_id)

### boardtype 테이블
 게시판 종류 정보 저장(PRIMARY KEY: type_id)

### board 테이블
 게시글 정보 저장(PRIMARY KEY: board_id)
